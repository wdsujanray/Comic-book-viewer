## 📖 About The Project

Comixo is a modern, fully responsive front-end website for a digital comic book company. It showcases the brand's collection, team, pricing plans, user reviews, and blog. The design focuses on an immersive visual experience for comic and manga enthusiasts.

**Key Features:**
- 🎨 Hero section with video modal and animated marquee
- 📊 Dynamic counters for user stats and experience years
- 🧑‍🎤 Team member profiles with social links
- 💳 Interactive pricing plans with a "Recommended" highlight
- ⭐ User review section with star ratings
- 📝 Trending comic news blog grid
- 🔍 Off-canvas search menu with sample comics
- 📱 Fully responsive design (mobile, tablet, desktop)

---

## 🛠️ Built With

- **HTML5** - Semantic markup
- **CSS3** - Custom styling & animations
- **Bootstrap 5** - Responsive grid & components
- **jQuery** - DOM manipulation & event handling
- **Slick Carousel** - Smooth sliders
- **Venobox** - Lightbox & video modals
- **Counter-Up / Waypoints** - Number animations
